# 🎓 Student Performance AI Dashboard

🚀 A Machine Learning powered web application that predicts student performance and provides actionable insights using an interactive Streamlit dashboard.

---

## 🔍 Problem Statement

Understanding student performance is critical for improving academic outcomes.
This project analyzes key factors such as study hours, attendance, sleep patterns, and previous scores to predict future performance.

---

## 💡 Solution

This system uses Machine Learning to:

* Predict student scores 📊
* Visualize key influencing factors 📈
* Provide smart recommendations 🤖

---

## 🧠 Key Features

* ✅ Real-time prediction using ML model
* ✅ Interactive dashboard built with Streamlit
* ✅ Feature-based performance analysis
* ✅ Compact data visualizations
* ✅ Personalized improvement tips

---

## ⚙️ Tech Stack

| Category         | Tools Used    |
| ---------------- | ------------- |
| Programming      | Python        |
| Data Analysis    | Pandas, NumPy |
| Machine Learning | Scikit-learn  |
| Visualization    | Matplotlib    |
| Dashboard        | Streamlit     |

---

## 📊 Model Details

* Algorithm Used: **Random Forest Regressor**
* Input Features:

  * Study Hours
  * Attendance
  * Previous Scores
  * Sleep Hours
  * Internet Usage
  * Extracurricular Activities
  * Parent Education Level
